## [1.0.0] - 2025-11-19

### Initial Release

#### Added
- **Line Chart** with 10+ variants
  - Simple line chart
  - Multi-series support
  - Curved lines (BÃ©zier)
  - Dashed lines
  - Area fill
  - Custom dot styling
  - Reference lines
  - Negative values support
  - Connect nulls option
  - Interactive features (zoom, pan, crosshair)

- **Bar Chart** with 6+ variants
  - Simple bar chart
  - Stacked bars
  - Grouped bars
  - Mixed bar types
  - Positive & negative values
  - Biaxial (dual Y-axis)

- **Pie Chart** with 3+ variants
  - Standard pie chart
  - Donut chart
  - Two-level hierarchical pie
  - Responsive design

- **Area Chart** with 9+ variants
  - Simple area
  - Stacked area
  - Percent area (100% stacked)
  - Cardinal (smooth) area
  - Connect nulls
  - Fill by value
  - Synchronized charts

- **Scatter Chart** with 6+ variants
  - Simple scatter
  - Bubble chart
  - Joint line + scatter
  - Grid cells
  - With labels
  - 3D effect

- **Radar Chart** with 2+ variants
  - Simple radar
  - Specified domain radar

- **Composed Chart** with 8+ variants
  - Line + Bar + Area combination
  - Multiple chart types in one view
  - Banded chart
  - Scatter + line of best fit
  - Target price chart
  - Vertical layout

- **Radial Bar Chart**
  - Concentric circular bars
  - Custom angle ranges
  - Background fills
  - Label positioning

- **TreeMap**
  - Hierarchical data visualization
  - Squarified layout algorithm
  - Nested rectangles
  - Custom colors per node
  - Depth-based styling
